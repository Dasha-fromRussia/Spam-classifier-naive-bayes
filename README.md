# Spam-classifier-naive-bayes
Классификация спам-сообщений с использованием наивного байесовского классификатора

## Описание
Проект по курсу «Интеллектуальный анализ больших данных». 
Реализована классификация SMS-сообщений на спам и не-спам с использованием наивного байесовского классификатора.

## Файлы
- `main.py` — основной код: предобработка, обучение модели, оценка
- `dashboard.py` — интерактивный дашборд на Dash/Plotly
